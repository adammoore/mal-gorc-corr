To import into Google Sheets:

  1. Download the Excel file using the button above
  2. Open Google Sheets and create a new spreadsheet
  3. Go to File → Import → Upload
  4. Select the downloaded Excel file
  5. Choose "Replace spreadsheet" and click "Import data"
Tip: The Excel file includes cell comments with detailed descriptions that will be preserved in Google Sheets.

About This Tool

This tool provides an interactive view and export functionality for the GORC-MaLDReTH correlation mapping table. The mapping demonstrates how Global Open Research Commons (GORC) infrastructure services complement and enable the MaLDReTH Research Data Lifecycle stages.

Key Features:
  • Clean correlation matrix with hover tooltips for descriptions
  • Excel export with formatted sheets and cell comments
  • CSV export for data analysis
  • Google Sheets compatible format
  • Analysis summary and key findings
